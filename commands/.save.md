---
name: save
description: Save current session context with optional note
---

Create a session save file in `.claude/history/` with this format:

**Filename:** `YYYY-MM-DD_HHMMSS_[description].md` (use current timestamp and description from $ARGUMENTS, or just timestamp if no description provided)

**File content:**
```markdown
# Session Save
**Date:** [current date and time]
**Repo:** [current git repo name or folder name]
**Description:** [description from $ARGUMENTS or "untitled session"]

## Current Work
[What we're currently working on]

## Technical Decisions
[Key architecture, library, and pattern choices made]

## Files Modified
[List of files changed and why]

## Progress
**Completed:**
[What's done]

**Remaining:**
[What's left to do]

## Important Context
[Any other critical information needed to resume]

## Next Steps
[Immediate next actions]
```
